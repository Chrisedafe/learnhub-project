# LearnHub

Frontend LMS project — browse courses, enroll, track progress, do lessons. Data lives in localStorage, no backend.

## Running it

```bash
npm install
npm run dev
```

Opens at [http://localhost:5173].



## Notes / things I'd do differently

- Would add skeleton loaders instead of the blank state on first load
- Dark mode only for now; light mode would just be swapping CSS vars
